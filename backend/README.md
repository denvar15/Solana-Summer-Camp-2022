# CryptoSteam backend, node.js version

## Setup

```bash
git clone https://github.com/denvar15/Solana-Summer-Camp-2022.git
cd backend
npm i --legacy-peer-deps
```

## Dev run

```bash
npm run watch-node
```

## Production run

```bash
npm run prebuild
npm run build
npm run start
```

## Accessing endpoints  

<http://localhost:8080/nft>
